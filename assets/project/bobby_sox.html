รายละเอียดโปรเจกต์

โปรเจกต์นี้เป็นการพัฒนาเว็บไซต์ร้านค้าออนไลน์โดยใช้ WordPress ร่วมกับปลั๊กอิน WooCommerce เพื่อสร้างระบบขายสินค้าแบบครบวงจร ตั้งแต่หน้าโชว์สินค้า ระบบจัดการคำสั่งซื้อ ไปจนถึงระบบชำระเงินออนไลน์ โดยใช้ XAMPP เป็นฐานข้อมูลสำหรับรันเว็บในเครื่อง จุดประสงค์ของโปรเจกต์นี้คือเพื่อเรียนรู้และเข้าใจการสร้างร้านค้าออนไลน์ตั้งแต่ระดับพื้นฐานจนถึงระบบหลังบ้านจริง พร้อมฝึกการตั้งค่าระบบ WooCommerce ให้รองรับการใช้งานจริง เช่น จัดการสต็อกสินค้า ค่าขนส่ง การรับออร์เดอร์ และการตั้งค่าธีมบน WordPress

สิ่งที่ทำในโปรเจกต์นี้

  • ความเข้าใจโครงสร้างเว็บไซต์ WordPress
  • ใช้งาน WooCommerce ในระดับสร้างระบบร้านค้าได้
  • ตั้งค่า Hosting ใน Localhost ด้วย XAMPP
  • จัดการฐานข้อมูล (phpMyAdmin)
  • เข้าใจระบบหลังบ้านของร้านค้าออนไลน์
  • ปรับแต่ง UI/UX ของร้านค้าบน WordPress Theme
  • ทำให้เว็บไซต์พร้อมต่อยอดไปเป็น MVP ร้านค้าออนไลน์จริง
