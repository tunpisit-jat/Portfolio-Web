Dream Shop Journey - เกมจำลองการบริหารชีวิตและการเงิน

รายละเอียดโปรเจกต์

Dream Shop Journey เป็นโปรเจกต์เกม 2D Top-down แนว Gamification + Life Simulation ที่ส่งเข้าประกวดในรายการ National Software Contest (NSC) 2025 โดยทีมของเราได้ผ่านเข้ารอบแรกและได้รับทุนสนับสนุนจำนวน 2,000 บาท เพื่อใช้พัฒนาต้นแบบเกม แนวคิดหลักของเกมคือการให้ผู้เล่นรับบทเป็นเด็กผู้ชายหรือเด็กผู้หญิงที่ต้องเริ่มต้นชีวิตจากศูนย์ ทำงานส่งอาหาร เก็บเงิน วางแผนชีวิต ด้วยการจัดการพลังของตัวละครให้สมดุลกับการทำงาน และบริหารร้านค้าของตัวเอง เพื่อทำตามความฝันในการเปิดร้านและขยายสาขาให้เติบโต

สิ่งที่ทำในโปรเจกต์นี้

  • ออกแบบระบบเกม (Game Design Document) ครอบคลุมระบบชีวิต การเงิน ร้านค้า และเหตุการณ์
  • ออกแบบ Core Gameplay Loop
  • ออกแบบ UX/UI และ Flow เกมบน Figma
  • สร้าง Prototype แนวคิดเกม พร้อมรูปแบบการเล่นหลัก
  • วิเคราะห์สมดุลเกม (Balance Game Economy)
  • จัดทำเอกสาร NSC เพื่อส่งเข้าประกวด
